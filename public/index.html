Help & Usage

Getting Started

Select a tool from the Quick Launch section. Tools marked as Ready are installed and available.

AI Tools Authentication

  • Gemini CLI: Run gcloud auth application-default login --no-launch-browser
  • Claude Code: Set ANTHROPIC_API_KEY in environment
  • Grok CLI: Run grok login or set XAI_API_KEY
  • GitHub Copilot: Run gh auth login

Terminal Tips

  • Use Ctrl+C to interrupt running processes
  • Type exit or press Back to Launcher to return
  • Terminal sessions persist while you stay connected
  • Mobile keyboards work with all shortcuts and commands

Workspace

Your projects are stored in the workspace directory. Each AI tool maintains its own configuration and authentication state.
